    What does "$(document).ready(function(){})" mean

    * "$()" says jQuery is about to begin. * Putting document between the parentheses tells usthat we're about to work our magic on the HTML document itself. * ".ready();" is a function, or basic action, in jQuery. It says "hey, I'm going to do stuff as soon as the HTML document is ready!" * Whatever goes in ".ready()'s" parentheses is the jQuery event that occurs as soon as the HTML document is ready. * "ready()" function ends with a semicolon to mark that we have done giving out query. "For making the jQuery object we need to give the exact selector as we used to give in css."

  • 3. adding a "p" after the button which has been clicked
    • 
[js]
      $(document).ready(function() {
          $('button').on('click', function() {
              var price = $('
    From $399.99
    ');
              // Add the price as a sibling after button
              $(this).after(price); // note the 'after' method
              $(this).remove();
          });
      });
[js]
    but the above code is fragile, as the result will be different when the"button" element will be at some other place.

                $(this).remove();
            });
        });
[js]
    the * closest * function finds the the parent which is 'closest' to the element.
